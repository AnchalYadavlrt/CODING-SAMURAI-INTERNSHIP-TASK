# Titanic Survival Prediction using Logistic Regression

## 📌 Description

This project uses the Titanic dataset to predict passenger survival using the Logistic Regression algorithm. The dataset is cleaned, preprocessed, and used to train a machine learning model for binary classification.

## 📂 Dataset

- Titanic Dataset (`train.csv`)
- Source: Kaggle Titanic Dataset
- **Download Link:** https://www.kaggle.com/c/titanic/data
- 
## 🛠️ Technologies Used

- Python
- Pandas
- Scikit-learn

## 📊 Features

- Data Cleaning
- Missing Value Handling
- Feature Encoding
- Train-Test Split
- Logistic Regression Model
- Model Evaluation
- Passenger Survival Prediction

## ▶️ How to Run

1. Download the Titanic dataset (`train.csv`).
2. Place it inside the project folder.
3. Install required libraries:

```bash
pip install -r requirements.txt
```

4. Run:

```bash
python main.py
```
