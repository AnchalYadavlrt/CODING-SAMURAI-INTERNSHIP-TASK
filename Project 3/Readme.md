# Titanic Dataset - Exploratory Data Analysis (EDA)

## 📌 Description

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand the data, identify missing values, analyze passenger information, and discover survival patterns using different visualizations.

## 📂 Dataset

- Titanic Dataset (`train.csv`)
- Source: Kaggle Titanic Dataset

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Features

- Loaded and explored the dataset
- Checked dataset information
- Identified missing values
- Filled missing Age values
- Generated summary statistics
- Visualized passenger demographics
- Analyzed survival patterns
- Created histograms, count plots, box plots, and correlation heatmap

## ▶️ How to Run

1. Download the Titanic dataset (`train.csv`).
2. Place the dataset in the project folder.
3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Run the program:

```bash
python main.py
```
