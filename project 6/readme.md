# Social Media Sentiment Analysis

## 📌 Description

This project performs sentiment analysis on social media tweets using the Sentiment140 dataset. The tweets are cleaned, converted into numerical features, and classified as Positive or Negative using Logistic Regression.

## 📂 Dataset

- Sentiment140 Dataset
- File: `training.1600000.processed.noemoticon.csv`
- **Download Link:** https://www.kaggle.com/datasets/kazanova/sentiment140
## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📊 Features

- Text Cleaning
- Feature Extraction
- CountVectorizer
- Logistic Regression
- Train-Test Split
- Model Evaluation
- Sentiment Prediction
- Data Visualization

## ▶️ How to Run

1. Place the dataset in the project folder.
2. Install required libraries:

```bash
pip install -r requirements.txt
```

3. Run:

```bash
python main.py
```
